# Automated Notification Workflow

This workflow checks for new updates and sends notifications.

1. Install dependencies (if not already installed)
// turbo
```bash
npm install
```

2. Run the notification checker script
// turbo
```bash
node scripts/check-updates.js
```

3. View the logs
```bash
cat logs/latest.log
```
